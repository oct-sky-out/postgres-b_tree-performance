### Page Split Impact

- **Time-based IDs**: ~1.5x splits (UUIDv7, ULID, KSUID)
- **Pattern IDs**: ~1.3x splits (Prefixed strings)
- **Random IDs**: ~2.5x splits (UUIDv4, Random strings)
